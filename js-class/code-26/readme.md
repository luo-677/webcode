# jQuery

### 什么是jQuery
jQuery是一个用于DOM操作和数据请求的JavaScript函数库
它提供了非常多的API易于操作DOM结构和数据请求。

jQuery当前版本 3.5.1 

### 历史版本
- 1.x 兼容性好 可以兼容至ie6 已经停止更新维护 最终版本1.12
- 2.x 过度版本 从开发到停止更新 只有半年 摒弃了兼容性代码 ie9+支持 最终版本2.2
- 3.x ie9+支持 体积更小 支持es6

版本迭代更新 核心函数和核心思想没有变
